OPÇÃO A: Automática (Recomendado - 20 min)
──────────────────────────────────────────

  $ cd /path/to/T3CK\ Core
  $ bash infrastructure/scripts/migrate-aws-to-gcp.sh
  
  Script vai:
  ✅ Validar Google Cloud SDK
  ✅ Criar bucket GCS
  ✅ Configura KMS encryption
  ✅ Rodar terraform init
  ✅ Migrar estado de S3 → GCS
  ✅ Validar tudo

  Depois:
  $ cd infrastructure/terraform
  $ terraform plan
  $ terraform apply

  Tempo total: ~1 hora


OPÇÃO B: Manual (Detalhadoado - 2 horas)
──────────────────────────────────────────

  Ver: infrastructure/MIGRATION_QUICKSTART.md (Option 2)
  
  Passo-a-passo manual:
  1. Criar bucket GCS
  2. Criar KMS key
  3. terraform init -upgrade
  4. Copiar state S3 → GCS
  5. terraform apply
