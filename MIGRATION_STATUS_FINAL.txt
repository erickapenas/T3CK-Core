## 🔄 O que foi alterado

### 1. **Terraform (infraestrutura)**
```diff
- provider "aws" { region = "..." }        ❌ Removido
- backend "s3" { bucket = "..." }          ❌ Removido
+ provider "google" { project = "..." }    ✅ Adicionado
+ backend "gcs" { bucket = "..." }         ✅ Adicionado
```
**Impacto**: Backend agora usa GCS com KMS encryption (90-day rotation)

### 2. **Dockerfile (backup-runner)**
```diff
- pip3 install --no-cache-dir awscli       ❌ Removido
- RUN apt-get install -y python3 python3-pip  ❌ Removido
  # gsutil já incluído em google/cloud-sdk:slim ✅
```
**Impacto**: Imagem 50% menor, sem dependências AWS

### 3. **Configuração (.env.example)**
```diff
- COGNITO_CLIENT_ID=...                    ❌ Removido
- COGNITO_USER_POOL_ID=...                 ❌ Removido
- PROD_BACKUP_BUCKET=s3://...              ❌ Removido
+ PROD_BACKUP_BUCKET=gs://...              ✅ Adicionado
```
**Impacto**: 100% GCP-only, Firebase Auth é autoridade única
