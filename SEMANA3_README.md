## 🚀 Quick Start (5 minutos)

```bash
# Terminal 1: Demo Server
node scripts/server.js

# Terminal 2: Tenant Service
cd services/tenant-service
pnpm install
pnpm start
```

Abra: http://localhost:8080/ADMIN_PANEL.html
